# Python: Strings

A string is a sequence of characters enclosed in quotes. Strings are used to store text such as names, messages, addresses, etc.

## Creating Strings

Using single quotes:
name = 'Nitin'

Using double quotes:
city = "Bangalore"

Using triple quotes (for multi-line strings):
message = """Welcome to Python.
This is a multi-line string."""

## Accessing Characters

Strings are indexed starting from 0.

text = "Python"

text[0]    # P
text[1]    # y
text[5]    # n

Negative indexing starts from the end.

text[-1]   # n
text[-2]   # o

## String Slicing

Syntax:
string[start:end:step]

Example:
text = "Python"

text[0:3]    # Pyt
text[2:6]    # thon
text[:4]     # Pyth
text[2:]     # thon
text[:]      # Python
text[::2]    # Pto
text[::-1]   # nohtyP (Reverse)

## String Concatenation

Joining two or more strings using the + operator.

first = "Hello"
second = "World"

result = first + " " + second
print(result)

Output:
Hello World

## String Repetition

Use the * operator to repeat a string.

text = "Hi "

print(text * 3)

Output:
Hi Hi Hi

## Useful String Functions

text = "python programming"

len(text)               # Length of string
text.upper()            # PYTHON PROGRAMMING
text.lower()            # python programming
text.title()            # Python Programming
text.capitalize()       # Python programming
text.strip()            # Removes leading/trailing spaces
text.replace("python", "Java")
text.split()            # Splits into a list
"-".join(["a", "b"])    # a-b

## Checking Strings

text = "Python"

text.startswith("Py")   # True
text.endswith("on")     # True
text.isalpha()          # True
text.isdigit()          # False
text.isalnum()          # True

## Finding Characters

text = "Python Programming"

text.find("Pro")        # 7
text.index("P")         # 0
text.count("m")         # 2

## Membership Operators

text = "Python"

"Py" in text        # True
"Java" in text      # False

"Java" not in text  # True

## String Formatting

Using f-strings (Recommended)

name = "Nitin"
age = 23

print(f"My name is {name} and I am {age} years old.")

Output:
My name is Nitin and I am 23 years old.

Using format()

print("My name is {}.".format(name))

## Escape Characters

\n    New line
\t    Tab
\\    Backslash
\'    Single quote
\"    Double quote

Example:

print("Hello\nWorld")

Output:
Hello
World

## Strings are Immutable

Strings cannot be changed after they are created.

text = "Python"

# Invalid
text[0] = "J"

This will raise an error.

To modify a string, create a new one.

text = "J" + text[1:]

Output:
Jython

## Summary

- Strings store text.
- Strings are enclosed in single, double, or triple quotes.
- Indexing starts from 0.
- Negative indexing starts from -1.
- Slicing extracts parts of a string.
- Strings are immutable.
- Use built-in methods like upper(), lower(), replace(), split(), and join().
- f-strings are the preferred way to format strings.















## Long (Multi-line) Strings

Triple quotes (`'''` or `"""`) are used to create multi-line strings. They allow the string to span multiple lines without using `\n`.

Using triple double quotes:

message = """
Hello,
Welcome to Python.
Have a great day!
"""

print(message)

Output:
Hello,
Welcome to Python.
Have a great day!

Using triple single quotes:

text = '''
Python is easy to learn.
Practice every day.
'''

print(text)

Output:
Python is easy to learn.
Practice every day.

Common Uses:
- Writing multi-line text.
- Creating long messages.
- SQL queries.
- Documentation (Docstrings).

Example:

query = """
SELECT *
FROM students
WHERE age > 18;
"""

print(query)
