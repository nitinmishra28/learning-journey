# Python: Formatted Strings

Formatted strings allow you to insert variables or expressions directly into a string. They make your code cleaner, more readable, and easier to maintain.

Python provides three common ways to format strings:
1. f-strings (Recommended)
2. format() method
3. % formatting (Older style)

## 1. f-Strings (Recommended)

Prefix the string with `f` and place variables or expressions inside curly braces `{}`.

Example:

name = "Nitin"
age = 23

print(f"My name is {name} and I am {age} years old.")

Output:
My name is Nitin and I am 23 years old.

### Expressions in f-Strings

You can also perform calculations inside the braces.

x = 10
y = 20

print(f"Sum = {x + y}")

Output:
Sum = 30

Example:

price = 100
quantity = 3

print(f"Total Price = {price * quantity}")

Output:
Total Price = 300

## 2. format() Method

The `format()` method replaces placeholders (`{}`) with values.

Example:

name = "Nitin"
age = 23

print("My name is {} and I am {} years old.".format(name, age))

Output:
My name is Nitin and I am 23 years old.

### Using Named Arguments

print("Name: {n}, Age: {a}".format(n="Nitin", a=23))

Output:
Name: Nitin, Age: 23

## 3. % Formatting (Old Style)

This is the oldest way of formatting strings. It is still supported but is less commonly used.

Example:

name = "Nitin"
age = 23

print("My name is %s and I am %d years old." % (name, age))

Output:
My name is Nitin and I am 23 years old.

Common Format Specifiers:

| Specifier | Meaning |
|-----------|---------|
| %s | String |
| %d | Integer |
| %f | Float |

## Formatting Numbers

Limit decimal places:

pi = 3.14159265

print(f"{pi:.2f}")

Output:
3.14

Example:

price = 99.999

print(f"Price: {price:.1f}")

Output:
Price: 100.0

## Formatting with Width

num = 42

print(f"{num:5}")

Output:
   42

print(f"{num:05}")

Output:
00042

## Aligning Text

name = "Python"

Left Align:

print(f"{name:<10}")

Output:
Python

Right Align:

print(f"{name:>10}")

Output:
    Python

Center Align:

print(f"{name:^10}")

Output:
  Python

## Why Use f-Strings?

- Easy to read.
- Faster than older formatting methods.
- Supports variables and expressions directly.
- Recommended for modern Python (Python 3.6+).

## Summary

- Use **f-strings** for most formatting tasks.
- Use **format()** when needed for compatibility or named placeholders.
- **% formatting** is the older style and is generally not recommended for new code.
- You can format numbers, align text, and evaluate expressions directly inside f-strings.
