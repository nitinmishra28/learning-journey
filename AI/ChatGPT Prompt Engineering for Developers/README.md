# ChatGPT Prompt Engineering for Developers

Instructor:
- Andrew Ng
- Isa Fulford (OpenAI)

This course teaches how to effectively communicate with Large Language Models using Prompt Engineering.

Rather than focusing on how LLMs are trained, this course focuses on using them effectively in real-world applications.

---

## Course Structure

1. Introduction
2. Guidelines for Prompting
3. Iterative Prompt Development
4. Summarizing
5. Inferring
6. Transforming
7. Expanding
8. Building Chatbots

---

## Learning Outcome

After completing this course, I will understand:

- Different types of LLMs
- Prompt Engineering
- Zero-shot Prompting
- Few-shot Prompting
- Chain of Thought Prompting
- Best Prompting Practices
- Building AI Applications using prompts
